Downhill Training

One-time setup: connect your Google Sheet so your logs sync across all your devices.

Step 1 — Create your Google Sheet

  1. Go to sheets.google.com and create a new blank spreadsheet
  2. Name it anything — e.g. "Downhill Training Logs"
  3. Leave it open, you'll need it for Step 2

Step 2 — Add the script

  1. In your Sheet, click Extensions → Apps Script
  2. Delete any existing code in the editor
  3. Paste the entire contents of TrainingApp.gs (the other file you downloaded)
  4. Click Save (disk icon), then click Deploy → New deployment
  5. Set type to Web app, set "Who has access" to Anyone, click Deploy
  6. Authorise when prompted, then copy the Web app URL you're given
